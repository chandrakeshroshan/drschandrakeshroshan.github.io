# Images Directory

## Profile Photo

Please add your professional profile photo here:
- Filename: `profile.jpg`
- Recommended size: 500x500 pixels or larger (square format)
- Format: JPG, PNG, or WebP
- The image should be a professional headshot with good lighting

## Favicon

Optionally add a favicon:
- Filename: `favicon.ico` or `favicon.png`
- Size: 32x32 pixels or 64x64 pixels

## Additional Images

You can add other images as needed for your website.
